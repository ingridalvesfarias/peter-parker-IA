# Peter IA - Assistente Virtual Web 🤖✨

O **Peter IA** é uma interface web moderna e minimalista para interação com inteligência artificial. Este projeto foca em oferecer uma experiência de usuário fluida (UI/UX) com suporte a comandos de texto e voz, utilizando tecnologias front-end de ponta.

## 🚀 Funcionalidades

* **Interface Dark Mode:** Design focado em conforto visual e estética moderna.
* **Comandos de Voz:** Integração preparada para captura de áudio via microfone.
* **Design Responsivo:** Adaptado para diferentes tamanhos de tela (Mobile First).
* **Chat Inteligente:** Layout de mensagens limpo e intuitivo.
* **Integração com API:** Conexão direta com modelos de linguagem (Gemini/Groq).

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização avançada com Flexbox/Grid e variáveis para o tema escuro.
* **JavaScript (ES6+):** Lógica de manipulação do DOM e consumo de APIs.
* **Web Speech API:** Para reconhecimento e síntese de voz no navegador.

## 📸 Preview da Interface

A interface conta com um campo de input estilizado, suporte a mensagens dinâmicas e ícones intuitivos para uma interação completa.
